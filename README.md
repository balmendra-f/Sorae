# Sorae

Sorae es una app Expo + Firebase para reflexión emocional conversacional. La
experiencia parte escuchando primero, construye memoria de forma progresiva y
permite que la persona borre el contexto guardado cuando quiera.

## Desarrollo

```bash
npm install
npm run start
```

## Validación local

```bash
npm run lint
npx tsc --noEmit
npm --prefix functions run build
```
